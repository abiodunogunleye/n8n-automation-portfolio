# AI Customer Enquiry Classifier — Estate Agent

## What It Does

Automatically classifies incoming WhatsApp messages from tenants, landlords, and buyers into 5 categories:
- **Sales Enquiry** — property purchase/viewing interest
- **Rental Enquiry** — property rental/tenancy interest  
- **Complaint** — maintenance issues, repairs needed
- **Urgent** — flooding, fire, break-in, gas leak, power outage
- **Spam** — irrelevant/promotional messages

Each category is routed to the correct team email automatically. All messages logged to Google Sheets.
